Everything you need to generate references quickly

PoseSandbox focuses on speed, clarity, and reliability — the things that actually matter for artists.

Pose joints + props

Click a joint or prop and pose it with the gizmo. Use Rotate and Move modes to get exact silhouettes.

  • Joint selection + outline
  • Rotate snap + axis lock
  • Move translations

Save / Load (JSON)

Download pose.json files and load them later. Great for tutorials, commissions, and consistent characters.

  • Pose notes
  • Props stored with transforms
  • Works offline after cache

Pose Gallery

Save thumbnails directly in your browser. Click any thumbnail to instantly reload that pose.

  • LocalStorage persistence
  • Rename / delete / clear
  • Instant re-open after refresh

Export PNG reference

Export clean PNG images for your drawing workflow. Perfect for gesture practice and storyboard frames.

  • One-click export
  • Consistent lighting
  • Shadows for depth

Fast + simple

No backend required. PoseSandbox runs entirely on GitHub Pages, so it’s cheap, stable, and scalable.

  • No accounts required
  • No build tools
  • Instant load

Artist-first UX

Clear UI, minimal distractions, strong contrast, and keyboard shortcuts for speed.

  • 1/2/3 mode keys
  • F to focus, Esc to clear
  • Del deletes props

FAQ

Keep this simple. You’re removing friction so people actually click “Launch”.

Is PoseSandbox really free?
Yes. The core web tool is free. The monetization plan is to add optional Pro content later, without breaking the free experience.
Do I need an account?
No. PoseSandbox runs entirely in your browser. Saved gallery poses stay on your device.
Does this work on GitHub Pages?
Yes. That’s why PoseSandbox is built with pure HTML/CSS/JS + import maps.
How do people discover PoseSandbox?
SEO + social previews + content + community links. This landing page includes metadata for Google, Discord, Twitter, and a sitemap/robots setup you’ll add as files.
